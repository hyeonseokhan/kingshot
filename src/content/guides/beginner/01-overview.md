# 🗺️ 초보 성장 로드맵

무과금/소과금 플레이어를 위한 핵심 성장 가이드입니다.

## 한 줄 요약

- 상위 연맹에 빨리 들어가야 상자, 가속, 버프 차이로 성장 속도가 벌어진다
- VIP7과 경기장은 소과금/무과금 계정의 핵심 성장 축이다
- 망치, 장비, 상점 구매 우선순위를 알면 장기 성장 효율이 크게 좋아진다

## ✅ 이것만 기억하면 절반은 성공

1. **건설 대기 + 행렬 대기**는 항상 돌릴 것
2. **상위 1~3위 연맹**에 최대한 빨리 진입
3. **VIP 7**이 1차 목표
   (매일 레전드 공용 조각 1개 루틴)
4. **경기장**은 레전드 장비와 조각 수급 루트
5. **망치(영웅 장비)**는 장기전 핵심
6. **상점 효율**을 알면 같은 재화로 훨씬 멀리 감

## ❌ 초보가 가장 많이 무너지는 패턴

| 실수 | 결과 |
|------|------|
| 연맹을 아무 데나 오래 머무름 | 누적 손해 발생 |
| 이벤트마다 조금씩 소비 | 아무것도 못 먹음 |
| 영웅/장비/다이아 분산 투자 | 경기장 순위 정체 |
| 상점에서 보이는 대로 구매 | 장기 효율 악화 |

## 🔄 성장의 전체 흐름

성장 구조는 **순환형**입니다:

> **연맹**(상자/버프) → **경기장**(레전드 장비) →
> **망치**(마스터리) → **더 높은 경기장 순위** → 반복

### 3단계 실행 로드맵

| 단계 | 목표 |
|------|------|
| 1단계 | 도시센터 건설, 상위 연맹 진입 |
| 2단계 | VIP7 달성, 경기장 순위 상승 |
| 3단계 | 망치와 장비 마스터리로 장기 경쟁력 확보 |
