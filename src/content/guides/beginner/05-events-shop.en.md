# 🛒 Events & Shop Efficiency Guide

## Event play: short-event 1st-place strategy

### Core principles

- **Always claim every base reward**
- Don't aim at every event — concentrate on
  **short events you can actually win**
- Spreading shards / speed-ups / diamonds across everything
  ends up earning nothing

### Operating points

1. Stockpile resources in advance
   to time **short-event victories**
2. Analyze server competition and
   **commit fully only to events you can win**
3. Run on the "base rewards + selective 1st-place events" structure

## Shop Efficiency Guide

> Free-to-play / low-spend players should
> **buy what stays on the account**, not what's needed right now

### Priorities by shop

#### Wandering Merchant

| Item | Recommendation |
|------|--------|
| Resources | ⭐⭐⭐ efficient early/mid-game |
| VIP Points (discounted) | ⭐⭐⭐ priority buy |
| Direct diamond purchase | ⭐ not recommended after Castle Lv.30 |

#### Mystic Shop

| Item | Recommendation |
|------|--------|
| Exclusive weapon discount | ⭐⭐⭐ top priority |
| Refresh | Limited use (overuse drops efficiency) |

#### Arena Shop

| Item | Recommendation |
|------|--------|
| Legendary chest | ⭐⭐⭐ top priority |
| Mithril | ⭐⭐⭐ strong choice |
| Mastery / gear | situational |

#### VIP Shop

| Item | Recommendation |
|------|--------|
| Legendary universal shard | ⭐⭐⭐ must buy |
| Hammer | ⭐⭐⭐ great long-term efficiency |
| Enhancement materials | ⭐⭐ great long-term efficiency |
| Speed-up bundles | ⭐ low priority for free-to-play |

#### Champion Shop

| Item | Recommendation |
|------|--------|
| Gold Thread / Silk | Buy at the right ratio |
| Sketch items | Add when you have spare resources |

#### Sacred Sword Shop

| Item | Recommendation |
|------|--------|
| Heroic gear materials | ⭐⭐⭐ top priority |
| 5 gem materials | situational |

#### Mightiest Lord Shop

| Item | Recommendation |
|------|--------|
| Combat-power resources | ⭐⭐⭐ top priority |
| Heroic gear materials | ⭐⭐ next |

#### Skin Shop

- Long-term accumulation, low short-term efficiency
- Only meaningful if you collect **consistently without breaks**

#### Tide Shop

| Item | Recommendation |
|------|--------|
| Pet rare materials | ⭐⭐⭐ top priority |
| Medals / Supplements / Premium Records | optional |

#### Trial Shop

| Item | Recommendation |
|------|--------|
| Mithril | ⭐⭐⭐ top priority |
| Premium shards | ⭐⭐ |

#### Diamond Shop

- Normally: **not recommended**
- Exception: event response, last-minute shortage cover

### Shop purchase priority — summary

1. Arena Shop → **Legendary chest**
2. VIP Shop → **Legendary universal shard + Hammer**
3. Wandering Merchant → **Resources**
4. Trial Shop → **Mithril**
