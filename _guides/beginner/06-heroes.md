# 🦸 영웅/스킬 가이드 (초보용)

## 소과금 현실 목표

> **SR 4성 → 첫 스킬 Lv.5 + 병사 티어 올리기**

## 핵심 원칙

### 영웅 육성

- 무작정 많은 영웅을 키우지 말 것
- **실제로 자주 쓰는 영웅** 중심으로 자원 압축
- 전투력 숫자만 보지 말고
  **집결과 경기장에 쓰이는지**를 기준으로 판단

### 스킬 투자

- 집결 참여용 영웅은
  **첫 스킬(1번 스킬) 효율**이 가장 중요
- 1번 스킬 Lv.5 달성을 우선 목표로

### 자원 배분 기준

| 기준 | 투자 여부 |
|------|-----------|
| 집결에 자주 사용 | ✅ 우선 투자 |
| 경기장에서 활용 | ✅ 우선 투자 |
| 전투력만 높음 | ❌ 후순위 |
| 가끔만 사용 | ❌ 후순위 |

## 참고 가이드

- 곰사냥 고득점 → 이벤트 탭 참고
- 영웅별 상세 스펙은 추후 업데이트 예정
